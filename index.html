Dit is een voorbeeld van een alinea op een webpagina

background-image : Dit is een css code die zorgt dat je een achtergrond op je website kan voegen.

background-repeat : Dit zorgt ervoor dat je achtergrond afbeelding in stukjes gaat.

background-size : Dit verandert de grootte van je achtergrond afbeelding

color : Dit veranderd de kleur van wat je wilt bijvoorbeeld je tekst.

margin-left : Dit veranderd de verte naar links op je pagina.

margin-top : Dit zet ruimte boven een element. Dus hoe groter de waarde, hoe verder het element naar beneden wordt geduwd.
